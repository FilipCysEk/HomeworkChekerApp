<h1>CSVlib Dokumentacja</h1>
<h2>Opis</h2>
<p>
Biblioteka ta służy do czytania plików CSV.
 Czytanie następuje dynamicznie, czyli nie jest czytant cały plik na raz, tylko przy chęci pobrania wiersza 
 jest on doczytywany.
 </p>

<h2>Dostępne funkcje</h2>

```cpp
int main;

```

<h2> Ruby Example </h2>

```ruby
x = 5 + 6
puts "Hello World! #{x}"
```
